# Cryptocurrency Website

This project implements a Cryptocurrency Website using HTML, CSS, and JavaScript. It uses the CoinGecko free API to fetch the latest prices of cryptocurrencies and provides a user-friendly interface for viewing this data.

## Features

- **Latest Cryptocurrency Prices**: The website displays the latest prices of various cryptocurrencies.
- **API Integration**: The website uses the CoinGecko free API to fetch the latest cryptocurrency data.
- **Stylish Design**: The website is designed with CSS to provide a sleek and modern look.
- **Easy to Use**: The website provides a simple interface for viewing the latest cryptocurrency prices.
- **Cross-Browser Compatibility**: The feature works across all modern web browsers that support HTML5 and CSS3.

## Installation

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser to view the webpage.

## Usage

To use this website, simply navigate to the webpage. The latest prices of various cryptocurrencies will be displayed on the screen.
